**プライバシーポリシー**

サービス名: Cro-co

施行日: 2026年6月18日

本プライバシーポリシー（以下「本ポリシー」といいます）は、マッチングサービス「Cro-co」（以下「本サービス」といいます）における利用者の個人情報の取扱いについて定めるものです。

本サービスは、大学生が相互に安全にコミュニケーションできる場の提供を目的として運営されます。本サービスは、個人情報の取扱いについて、個人情報の保護に関する法律（以下「個人情報保護法」といいます）その他関係法令およびガイドラインを遵守し、適法かつ適切に管理します。

**第1条（個人情報取扱事業者）**

本サービスの個人情報取扱事業者は、本サービスを開発・運営する個人（以下「運営者」といいます）です。

運営者の氏名等の詳細につきましては、本ポリシー第15条に定めるお問い合わせ窓口を通じてご請求ください。運営者は、本人確認の手続を経たうえで、合理的な範囲で開示します。

**第2条（取得する個人情報の項目）**

運営者は、本サービスの提供にあたり、以下の項目を取得します。

**(1) 認証情報**

メールアドレス（@ecs.osaka-u.ac.jp ドメインのものに限る）

パスワード（暗号学的にハッシュ化された形式で保管）

**(2) プロフィール情報**

表示名、自己紹介文、ステータスメッセージ

学年、学部、学科、所属サークル、出身地、興味関心

プロフィール写真（最大6枚）。なお、写真は公開前に運営者による適切性の確認を実施します

性別、興味の対象

**(3) 本人確認情報（本ポリシー第4条に基づき特別に取り扱います）**

本名（氏名）

学籍番号

生年月日

学生証画像

**(4) 利用状況情報**

マッチング履歴、メッセージ送受信記録、メッセージへのリアクション

いいね、ブロック、通報、非表示、プロフィール閲覧の各履歴

**(5) 通知設定**

Webプッシュ通知の購読情報（エンドポイントURL、暗号鍵）

**(6) お問い合わせ情報**

お問い合わせを通じて送信された内容および対応の記録

**(7) 端末・アクセス情報**

IPアドレス、ブラウザ・端末情報

サービスワーカーおよびIndexedDBに保存されるキャッシュ情報（表示速度の向上のため）

**第3条（利用目的）**

運営者は、取得した個人情報を以下の目的のためにのみ利用します。

| **情報の種類** | **利用目的** |
| --- | --- |
| 本人確認情報（第2条第3号） | 利用者が在籍大学の現役学生かつ18歳以上であることの確認、不正登録の防止、利用規約違反者の再登録防止 |
| プロフィール情報（第2条第2号） | マッチング機能およびコミュニケーション機能の提供、投稿写真の利用規約適合性の確認 |
| 認証情報（第2条第1号） | アカウント管理、不正利用の検知 |
| 利用状況情報（第2条第4号） | サービス機能の提供、不正利用の検知および対応 |
| 通知設定（第2条第5号） | プッシュ通知の配信 |
| お問い合わせ情報（第2条第6号） | お問い合わせへの対応 |
| 端末・アクセス情報（第2条第7号） | セキュリティの確保、アクセス解析、サービス品質の向上 |

運営者は、利用目的を変更する場合、変更後の目的が変更前の目的と相当の関連性を有すると合理的に認められる範囲を超えないものとし、変更後は本ポリシーへの反映と利用者への通知または公表を行います。

**第4条（本人確認情報の特別取扱い）**

本ポリシー第2条第3号に定める本人確認情報については、以下のとおり特別に取り扱います。

**(1) 取得の目的**

本人確認情報は、利用者が在籍大学の現役学生かつ18歳以上であることの確認、および本サービスの安全性確保のためにのみ取得します。

**(2) 物理削除の時期**

学生証画像および本名・学籍番号・生年月日の平文データは、以下のタイミングで物理的に削除します。

審査において承認された場合: 承認時点から5日以内

審査において不承認となった場合: 不承認時点から31日以内

利用者が退会した場合: 退会手続の完了時点で即時

**(3) ハッシュ値の保持**

本名および学籍番号については、本サービスの利用者が(5)に該当しない限り、物理削除後も以下の目的のためにハッシュ値（元の情報を復元できない一方向的な変換結果）を1年間保持します。

不正行為・キャンペーン悪用等への対応

法令に基づく要請への対応（注: ハッシュ値は元の情報に復元できないため、個人の同定を必要とする捜査協力への対応には限界があります）

1年間の保持期間が経過した後は、ハッシュ値も物理的に削除します。

**(4) 削除実行の責任**

物理削除は、運営者が定期的に実行する自動処理によって実施されます。削除実行の記録は内部監査ログとして保持されます。

**(****5****) ****本サービスの利用者が利用規約もしくは法令に違反していた場合　**

本サービスの利用者が利用規約もしくは法令に違反していた場合、もしくは違反していたことが退会後1年以内に判明した場合、該当者の本名および学籍番号のハッシュ値を物理削除せず永久に保持できるものとします。

また、該当者のハッシュ値は再登録防止措置にのみに使用されるものとしますが、法令に基づく要請があった場合にはこの限りではありません。

**第5条（各情報の保存期間）**

取得する個人情報の保存期間は、以下のとおりです。

| **情報の種類** | **保存期間** |
| --- | --- |
| 学生証画像 | 審査承認後5日以内・不承認後31日以内・退会時即時削除 |
| 本名・学籍番号・生年月日（平文） | 学生証画像と同一タイミングで物理削除 |
| 本名・学籍番号のハッシュ値 | 退会後から1年間（その後物理削除）※利用規約、法令の違反者については第4条(5)に従います。 |
| プロフィール情報 | 退会時に即時削除 |
| メッセージ送受信記録・リアクション | 退会時に即時削除 |
| いいね・ブロック・通報・非表示・閲覧履歴 | 退会時に即時削除 |
| お問い合わせ情報 | 退会時に即時削除（対応完了後は随時整理） |
| 端末・アクセスログ（IPアドレス等） | 取得から最大90日（セキュリティ目的） |
| サービスワーカー・IndexedDBキャッシュ | ログアウト時または退会時に削除。ブラウザのキャッシュクリアによっても削除可能 |

法令により保管が義務付けられる記録については、当該義務の範囲内で必要な期間に限り保管します。

**第6条（個人情報の安全管理措置）**

運営者は、個人情報の漏洩、滅失または毀損を防止するため、以下の安全管理措置を講じます。

**(1) 技術的安全管理措置**

通信の暗号化（TLS 1.2以上）

データベースへのアクセス制限（認証・権限管理）

認証情報の暗号化保管

学生証画像の非公開ストレージでの管理

**(2) 組織的安全管理措置**

個人情報へのアクセスを、運営者および承認された管理者に限定

管理者による主要な操作の記録

**(3) 人的安全管理措置**

個人情報を取り扱う機器の適切な管理

**第7条（個人情報の第三者提供）**

運営者は、以下の場合を除き、あらかじめ利用者の同意を得ることなく、個人情報を第三者に提供しません。

1. 法令に基づく場合

2. 人の生命、身体または財産の保護のために必要であり、利用者の同意を得ることが困難な場合

3. 公衆衛生の向上または児童の健全な育成のために特に必要であり、利用者の同意を得ることが困難な場合

4. 国の機関もしくは地方公共団体またはその委託を受けた者からの法令に定める事務の遂行への協力依頼に応じる必要があり、利用者の同意を得ることにより当該事務の遂行に支障を及ぼすおそれがある場合

**第8条（業務委託先）**

運営者は、本サービスの提供にあたり、以下の事業者に業務の一部を委託しています。

| **委託先** | **委託する業務** | **所在国** |
| --- | --- | --- |
| Supabase Inc. | データベース・認証基盤・ファイルストレージの提供 | アメリカ合衆国 |
| Vercel Inc. | フロントエンドのホスティング | アメリカ合衆国 |
| Render Services, Inc. | バックエンドのホスティング | アメリカ合衆国 |
| Resend, Inc. | メールの配信 | アメリカ合衆国 |
| Google LLC | アクセス解析（Google Analytics） | アメリカ合衆国 |

Google Analytics の利用は、利用者の同意取得を前提として実施します（本ポリシー第10条参照）。

**第9条（個人情報の越境移転）**

本ポリシー第8条に記載のとおり、本サービスは個人情報の一部を日本国外（主としてアメリカ合衆国）に所在する委託先で保管または処理しています。

アメリカ合衆国における個人情報の保護に関する制度等の詳細は、個人情報保護委員会のウェブサイト（https://www.ppc.go.jp/）にてご確認いただけます。

**第10条（Cookieおよび類似技術の利用）**

本サービスは、以下のCookieおよび類似技術（localStorage、IndexedDB、Service Worker キャッシュ等）を利用します。

**(1) サービス提供に必須の技術**

ログイン状態の維持、利用者の設定の保存、サービスの表示速度向上のために利用します。これらを無効化した場合、本サービスの一部または全部が利用できなくなることがあります。

**(2) アクセス解析（Google Analytics）— 外部送信について**

運営者は、利用者の同意を得た場合に限り、Google LLC の提供する Google Analytics を利用し、本サービスの利用状況を解析します。同意した場合、利用者の端末から Google LLC に対し、次の情報が送信されます。

送信される情報の内容：Cookie 等の識別子、閲覧したページの URL、リファラー、IPアドレス、ブラウザ・端末情報、利用日時等

送信先（情報を取り扱う者）：Google LLC（アメリカ合衆国）

利用目的：本サービスのアクセス状況の分析およびサービス品質の向上

同意は登録画面の任意項目で取得し、同意しない場合 Google Analytics は読み込まれず、上記の送信は行われません。同意後も、ブラウザの設定または Google Analytics オプトアウト アドオン（https://tools.google.com/dlpage/gaoptout?hl=ja）により送信を停止できます。なお、本サービスの提供に必要不可欠な Cookie 等（ログイン状態の維持等）は本項の対象外です。

**(3) Webプッシュ通知**

マッチングやメッセージ受信の通知のために、Webプッシュ通知（VAPID方式）を利用します。プッシュ通知の購読解除は、ブラウザの設定画面、または本サービスの通知設定画面からいつでも行えます。

**第11条（未成年者の取扱い）**

本サービスは、利用者の年齢が18歳以上であることを登録時に確認します。確認は、生年月日の自己申告および学生証画像の目視審査の2段階で実施します。

万が一18歳未満の利用者からの個人情報の取得が判明した場合、運営者は当該利用者のアカウントを停止し、保有する個人情報を遅滞なく削除します。

**第12条（退会および個人情報の削除）**

利用者は、本サービス内のアカウント設定画面から、いつでも退会（アカウントの削除）を申し出ることができます。退会手続の完了時点で、以下のとおり個人情報を取り扱います。

**(1) 即時に物理削除する情報**

プロフィール情報、いいね・ブロック・通報・非表示・閲覧履歴、通知設定、お問い合わせ情報は、退会手続の完了時点で即時に物理的に削除します。

**(2) メッセージ送受信記録・リアクションの取扱い**

メッセージ送受信記録およびメッセージへのリアクションは、退会手続の完了時点で即時に物理的に削除します。

**(3) 退会後1年間保持した後に物理削除する情報**

本名および学籍番号のハッシュ値は、不正登録防止等の目的のため1年間保持します（本ポリシー第4条第3号参照）。1年経過後は物理的に削除します。

**(4) 退会後の他の利用者からの参照**

退会後、当該利用者のプロフィールおよび他の利用者との間で送受信されたメッセージ等の記録は、退会時点をもって他の利用者の画面から参照できなくなります。

**(5) 例外**

法令の定めにより一定期間の保管が義務付けられている記録、および紛争解決のため合理的に必要となる記録については、当該義務または必要性の範囲内で必要な最低限の期間に限り保管する場合があります。

**第13条（個人情報の漏洩等が発生した場合の対応）**

運営者は、個人情報の漏洩、滅失、毀損その他の安全の確保に係る事態が生じた場合、個人情報保護法第26条の定めに従い、以下の対応を行います。

1. 個人情報保護委員会への速やかな報告

2. 影響を受ける利用者への個別通知（通知が困難な場合は、本サービス上での公表）

3. 再発防止策の実施

**第14条（開示・訂正・利用停止等の請求）**

利用者は、運営者に対し、個人情報保護法に基づき、自己の個人情報について以下の請求を行うことができます。

1. 利用目的の通知

2. 開示（電磁的記録の提供を含む）

3. 内容の訂正、追加または削除

4. 利用の停止または消去

5. 第三者提供の停止

請求方法は、本ポリシー第15条に定めるお問い合わせ窓口までご連絡ください。運営者は、ご本人様確認のうえ、法令に定める期間内に対応します。

**第15条（お問い合わせおよび苦情申し出窓口）**

本ポリシーおよび個人情報の取扱いに関するお問い合わせ・苦情の申し出、ならびに第14条に定める請求は、以下までご連絡ください。

メールアドレス: support@crocoweb.jp

なお、本サービスは現在、個人情報保護法に基づく認定個人情報保護団体に加入していません。個人情報の取扱いに関して問題が解決しない場合は、個人情報保護委員会（https://www.ppc.go.jp/）にご相談いただくことも可能です。

**第16条（本ポリシーの改定）**

運営者は、法令の変更または本サービスの内容の変更等に応じ、本ポリシーを改定することがあります。

重要な改定を行う場合は、本サービス内での通知または利用者の登録メールアドレス宛ての通知により、改定内容および改定の効力発生日を事前にお知らせします。

利用者が改定の効力発生日以降に本サービスを利用した場合、改定後の本ポリシーに同意したものとみなします。

**附 則**

本ポリシーは、2026年6月18日に施行します。
